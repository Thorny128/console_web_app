How to use the Console to debug your code

The console is a very useful tool to debug your code. You can access it by right clicking on nowhere in the page, clicking inspect, and clicking Console. After certain parts in your code, putting console.log to say "Action #1 done" or "Link copied" will help debug your code if something goes wrong, and you don't know where. It will seem like a burden early on, but it may save you hours of debugging later on if you make a mistake. Take it from me - you want to be using the console.

Try opening the console now. You should see a name and a number there.

Now click on this button, and check the console after you do it.

Notice anything? Yup, events in the console. One example of using this in debugging is if the text didn't appear. You would know that something is wrong in the code regarding the button, and instead of debugging the whole program, you would just need to debug the button code. It saves so much time!

Try it yourself! Make some code and use the console to log events! Try using buttons, links, and more to trigger console events! Then, ask a parent or friend to mess up some part of the code, and watch the console help you debug!
